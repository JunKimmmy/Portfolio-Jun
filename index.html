    My Projects

    Bionic EVO: Prosthetic Arm

    September 2022 - December 2023

    Skills:
    C++, SOLIDWORKS, LaTeX, 3D Printing, Arduino IDE

    Components:
    STM32 Microcontroller, SG90 Servo Motor, Electromyography (EMG) Sensor, Potentiometer, Arduino UNO

    Project Description:
    The primary objective of this project is to design and implement an advanced prosthetic human arm that helps amputees regain functionality. The aim is to create a prosthetic arm that closely mimics the capabilities of a real human arm, allowing users to perform a wide range of daily activities and regain a sense of independence.

    Using a STM32 Microcontroller in conjunction with 5 servo motors and a electromyography (EMG) Sensor the device is controlled using muscle control via the EMG sensor, which then dictates the movement of the servos that contract the fingers of the prosthetic using string.

    I leveraged 75+ months expertise in C++ programming and circuitry design to create an efficient control system that would work seamlessly with the large quantity of components used, and an advanced input such as the EMG sensor.

    I harnessed 2 years of SOLIDWORKS experience skills via a mentor to create a functional 3D design for the prosthetic that was modeled based on meticulous measurements based on human anatomy and to house each component.

    BrailleEye: AI, Object Detection, Braille Label Maker

    Feburary 2024 - TBD

    Skills:
    C++, SOLIDWORKS, 3D Printing, Arduino IDE, Edgeimpulse

    Components:
    ESP32S3-CAM, Stepper Motor, Button, Braille Label maker

    Project Description:
    The project entailed the integration of a Braille label maker and ESP32 microcontroller, establishing seamless communication between the two components. Leveraging the ESP32's camera module, the system interfaced with an AI model trained manually via Edge Impulse, enabling real-time object detection. A sophisticated process was engineered to convert detected object names into Braille, meticulously translating each letter of the identified object's name into corresponding angles. This precise control mechanism facilitated the manipulation of a stepper motor, orchestrating the turning of the dial on the Braille label maker with accuracy and efficiency.

    Gyro Controlled Excavator Arm

    December 2021 - June 2023

    Skills:
    C++, SOLIDWORKS, 3D Printing, Arduino IDE

    Components:
    Arduino Micro, SG90 Servo Motor, MPU-6050 Accelerometer Gyroscope

    Project Description:
    Embarking on the Gyro Controlled Excavator project served as a dynamic learning journey, introducing me to SOLIDWORKS modeling, MPU-6050 Accelerometer Gyroscopes, and refining soldering skills. The project aimed to fuse these newfound skills into a cohesive robotic endeavor.

    Using SOLIDWORKS, I meticulously designed a miniature robotic arm with three servo motors, each governing a joint's movement. Soldering the Gyroscope to the microcontroller facilitated seamless integration, enhancing the arm's functionality.

    Programming the Gyro enabled precise motion interpretation, empowering the robotic arm to respond with fluidity and accuracy. This holistic process honed skills in CAD modeling, soldering, and programming, underscoring the project's role as a catalyst for continuous learning and exploration in robotics.
